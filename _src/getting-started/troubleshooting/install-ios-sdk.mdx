# Install iOS SDK

- Launch Xcode
- Accept SDKs Agreement

<img src={useBaseUrl("img/ios-sdk-1.png")} style={imgStyle} />

- Select iOS platform and optionally other
- Click `Download & Install`

<img src={useBaseUrl("img/ios-sdk-2.png")} style={imgStyle} />

<details>
<summary>Don't see the "sdk install prompt"?</summary>

- Select "Xcode" > "Settings"

<img src={useBaseUrl("img/ios-sdk-2-1.png")} style={imgStyle} />

- Make sure that iOS is installed. Click "Get" if it is not installed.

<img src={useBaseUrl("img/ios-sdk-2-2.png")} style={imgStyle} />

</details>

- Wait for SDK's to install

<img src={useBaseUrl("img/ios-sdk-3.png")} style={imgStyle} />

- Dismiss `What's new in Xcode`

<img src={useBaseUrl("img/ios-sdk-4.png")} style={imgStyle} />
